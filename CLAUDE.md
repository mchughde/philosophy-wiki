# Philosophy Wiki — Project Instructions

## Source Material Discipline

When drafting or redrafting encyclopedia entries, use **only** the source material provided in the `Source material/[Philosopher]/` subfolders. Do not draw on background training knowledge to fill gaps.

**Rules:**
- If a fact is not in the provided sources, say so rather than supplying it from memory.
- Cite which source supports each substantive claim if asked.
- Flag anything uncertain with a note like: "I'm not sure this is in your sources."
- Do not import details that feel familiar but aren't traceable to the provided files.

**Why this matters:** Background knowledge about philosophy is deep and can slip in unnoticed — as happened with the detail "Megara" (not in any provided source), "narrowly escaped serious harm" (not in any provided source), and an error about Epicureans (directly contradicted by the Stanford source).

## Content Workflow

Source material goes in: `Source material/[Philosopher]/` (scraped MD files from Britannica, IEP, Stanford)  
Entries are stored in: `entries.js` (ENTRIES array)  
Article structure: Biographical Details / Key Ideas / Major Works
